SUPPORT & TROUBLESHOOTING
================================================================================

If fraud check returns null:
  1. Check BD_COURIER_API_KEY is set
  2. Verify API key is valid (test with curl)
  3. Check console for [FraudCheck] error messages
  4. Verify network connectivity to api.bdcourier.com

  1. Check report dates (should decay over time)
  2. Verify severity ratings are accurate
  3. Review weight calculation logic

If timeouts occur:
  1. Check network latency to BD Courier API
  2. Consider increasing timeout (10s/15s)
  3. Monitor API availability
